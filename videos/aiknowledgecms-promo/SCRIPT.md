# Narration Script

AIKnowledgeCMS is not a normal admin panel. It is a CMS operated by an agent loop.

Every tick senses the site, researches sources, triages issues, creates content, verifies quality, and distributes the result.

The loop is declared in one Loopfile: KPIs, cadence, budgets, gates, escalation, and adapters.

That means growth is not a manual task. It becomes a measurable, auditable operating cycle.

AIKnowledgeCMS is already dogfooding itself: live loop reports, generated articles, and gated publishing are running on the reference site.

There is no admin panel. There is a loop.
